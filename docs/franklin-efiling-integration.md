# Franklin County Common Pleas (Criminal) E-Filing Readiness Notes

This note is for preparing a **court-facing packet** and internal product readiness checks.

## Core filing format requirements

- Filings should be submitted as **PDF** files.
- Proposed orders should be provided as **.docx** files.

## File size limits

- **5 MB per file** maximum.
- **25 MB total transmission** maximum across one submission.

## Document style expectations

- Use **Arial** or **Times New Roman**.
- **12 pt minimum** font size.

## Signature format

- Use a conformed electronic signature format:
  - **`/s/ Full Name`**

## Filer limitations to surface in UX

- Add a clear limitation check for **self-represented incarcerated filers**.
- If user is incarcerated and self-represented, flag for alternative filing path / clerk guidance.

## Product-language guardrails

- Use the phrase **"E-filing-ready packet"**.
- Do **not** describe functionality as automatic filing.
- RecordPathAI should **not** claim to file directly with the court.
